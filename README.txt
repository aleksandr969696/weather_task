Для работы программы необходимо установить следующие библиотеки: pandas_gbq, requests, pyarrow.

Запуск через конамндную строку осуществляется либо без параметров, либо с одним, двумя или тремя параметрами.
Параметр --table (-t) отвечает за название таблицы (по умолчанию 'weather_table'), 
параметр --dataset (-d) - за название набора данных (по умолчанию 'weather_dataset'),
а параметр --key (-k) отвечает за название файла с ключом доступа к BigQuery (по умолчанию 'key.json').

Формат команды: python <filename.py> [--table <tablename>] [--dataset <datasetname>] [--key <keyname.json>].

и прописать в параметрах путь к этому файлу. 

Таблицы и наборы данных создаются автоматически, если таковых не имеется. 
При создании таблицы выдаёт предупреждение, но всё выполняет правильно.
